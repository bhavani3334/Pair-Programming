# Software Engineering – Pair Programming Task (Agile Methodology)

## Project Overview

This repository contains Python programs developed as part of the **Software Engineering Pair Programming Task** using the **Agile methodology**.

The programs were created with a focus on:

* Simple and understandable code
* Collaboration through pair programming
* Incremental development using Agile practices
* Testing and improving solutions step by step

## Technologies Used

* Python
* Git
* GitHub

## Programs Included

1. Stack Operations (Menu Driven Program)
2. GCD of Two Numbers
3. String Palindrome Check
4. Pattern Matching
5. Pair of Numbers Within a Given Limit

## Agile Approach Followed

* Requirement understanding
* Task division between pair members
* Coding and implementation
* Testing and debugging
* Final review and completion

## Objective

To practice software development using pair programming techniques and Agile principles while improving coding and collaboration skills.

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run any Python file using:

```bash
python filename.py
```

* Understanding Agile workflow

* Python programming practice
